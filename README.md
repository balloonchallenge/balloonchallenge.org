## Setting up the project


    git clone https://github.com/balloonchallenge/balloonchallenge.org.git

    # get rails gems

    bundle install

    # set up the database

    rake db:create
    rake db:migrate

## Running the project

    # start the server
    rails s
    # run tests/livereload
    guard


Write tests if you're doing any kind of logic.
